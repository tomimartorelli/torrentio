# Torrentio - Plataforma de Juegos

## 🚀 Despliegue en Producción

### 🌐 Demo
- **Frontend**: https://torrentio.vercel.app
- **Backend**: https://torrentio-backend.vercel.app
- **Base de Datos**: MongoDB Atlas

### 🛠️ Stack Tecnológico
- **Frontend**: React 18, Vite, TailwindCSS
- **Backend**: Node.js, Express, MongoDB
- **Autenticación**: JWT
- **Despliegue**: Vercel

### ✨ Características Principales
- 🎮 Catálogo de juegos con búsqueda y filtros
- 👥 Sistema de autenticación de usuarios
- 📊 Panel de administración
- 📱 Diseño responsive y tema claro/oscuro
- ⚡ Rendimiento optimizado

### 📦 Arquitectura
- **Frontend**: SPA con React Router
- **Backend**: API REST con Express
- **Database**: MongoDB con Mongoose
- **Storage**: Multer para uploads de imágenes

### 🔧 Configuración del Proyecto
1. Clonar el repositorio
2. Configurar variables de entorno (.env)
3. Instalar dependencias: `npm install`
4. Iniciar frontend: `npm run dev`
5. Iniciar backend: `npm start`

### 🌟 Aspectos Destacados para Portfolio
- Diseño minimalista y profesional
- Manejo de estados complejos con React Context
- Sistema de temas claro/oscuro
- Optimización de rendimiento
- Código limpio y mantenible
